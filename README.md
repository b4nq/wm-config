## installation
Clone this repository
```bash
git clone https://github.com/bankubanku/arch-config.git 
```
change to the cloned repository
```bash
cd arch-config
```
make setup.sh executable
```bash
chmod +x setup.sh
```
execute setup.sh
```bash
./setup.sh
```
